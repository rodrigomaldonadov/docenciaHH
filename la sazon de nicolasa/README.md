# Proyecto para aprender Git

Este proyecto fue creado para entender y practicar el uso de **Git** en un entorno real de trabajo academico.

## Objetivo

Aprender a usar Git para llevar control de versiones, registrar cambios y trabajar de forma ordenada en el desarrollo de un proyecto.

## Que se practica

- Inicializar y conectar un repositorio.
- Revisar cambios con `git status`.
- Agregar archivos con `git add`.
- Guardar avances con `git commit`.
- Subir cambios al remoto con `git push`.

## Flujo basico

```bash
git status
git add .
git commit -m "Describe tu cambio"
git push
```

## Nota

La idea principal de este proyecto es reforzar la practica constante de Git durante el curso.
